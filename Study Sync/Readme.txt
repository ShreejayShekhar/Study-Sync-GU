StudySync - Weekly Planner & Timetable Assistant
-------------------------------------------------

🔹 Description:
StudySync is a Java Swing-based GUI application designed to help students manage their weekly academic workflow. It allows them to add, track, and complete weekly tasks, while also managing a personal class timetable. The dashboard gives a clear overview of both.

🔹 Features:
✅ Add weekly tasks with multiple assigned days  
✅ Mark tasks complete for a selected day via dropdown  
✅ Add timetable entries with both start and end time  
✅ View all tasks and classes in a clear, scrollable dashboard  
✅ Dynamic refresh when switching to Dashboard tab  
✅ All UI built in Java Swing with good UX practices
